# PeakGuide

PeakGuide is a full-stack web application for exploring the Crown of Polish Mountains (Korona GÃ³r Polski).

## Tech stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: PostgreSQL
- Auth: JWT
- Deployment: Render (API + DB), Netlify (Frontend)

## Features (MVP)
- Multilingual content (PL / EN)
- Peaks and mountain ranges
- Detailed peak pages
- User status: planned / done
- Admin CRUD for peaks and trails
